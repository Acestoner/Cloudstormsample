Introduction

CloudStorm is the global edge-to-cloud company built to transform your business. How? By helping you connect, protect, analyze, and act on all your data and applications wherever they live, from edge to cloud, so you can turn insights into outcomes at the speed required to thrive in today's complex world.

Accelerating your business outcomes with comprehensive solutions, from edge to cloud.

Control and harness data across edge to cloud

Take the CloudStorm GreenLake for a no-cost, no-obligation test drive. This guided, hands-on experience allows you to explore cloud services in a live production environment.

Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (homebrew)

$ brew install CloudStorm-cli

NPM

$ npm install CloudStorm-cli

Yarn

$ yarn install CloudStorm-cli
